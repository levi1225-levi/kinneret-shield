USAGE INSTRUCTIONS:
===================

1. Install dependencies:
   flutter pub get

2. Generate code:
   dart run build_runner build

3. Configure environment:
   Update lib/core/constants/app_constants.dart with:
   - Supabase URL & key
   - Google OAuth credentials
   - Backend API URL
   - WebSocket endpoint

4. Run application:
   flutter run

5. Build for release:
   flutter build ios/apk/web

DATABASE SCHEMA INTEGRATION:
============================

The app expects these endpoints from NestJS backend:

Auth:
  POST /auth/google/signin
  POST /auth/register
  GET /auth/me
  POST /auth/verify

Attendance:
  GET /attendance
  GET /attendance/:id
  GET /attendance/today
  POST /attendance

Rooms:
  GET /rooms
  GET /rooms/:id
  GET /rooms/:id/occupancy

Devices:
  GET /devices
  GET /devices/:id
  GET /devices/status/online

Alerts:
  GET /alerts
  GET /alerts/active
  GET /alerts/unread/count
  PUT /alerts/:id/resolve

Emergency:
  GET /emergency/active
  GET /emergency/events
  POST /emergency/trigger
  PUT /emergency/events/:id/resolve
